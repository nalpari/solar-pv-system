# 폴리곤 UX 개선 디자인

## 목적

크롭 팝업 내 폴리곤 에디터의 편의성 향상. PC와 터치 기기 모두 지원.

## 기능 범위

### 1. 그리기 중 Undo

- 마지막으로 찍은 점을 취소
- 입력: 캔버스 우하단 플로팅 버튼 / PC 우클릭
- 점이 1개 이상일 때만 플로팅 버튼 표시

### 2. 선택/이동 모드에서 폴리곤 조작

선택/이동 모드에서 폴리곤 클릭 시 툴팁(이동 / 삭제 / 점 편집) 표시.

#### 이동

- 툴팁에서 "이동" 선택 후 폴리곤 내부 드래그로 통째 이동
- 이동 중 반투명 처리
- 드래그 종료 시 위치 확정, idle 복귀

#### 삭제

- 툴팁에서 "삭제" 선택 시 즉시 제거

#### 점 편집

- 툴팁에서 "점 편집" 선택 시 꼭짓점에 원형 핸들 표시 (반지름 12px, 터치 고려)
- 변(edge) 중간에 작은 + 핸들 (점 추가용)
- 핸들 드래그 = 점 이동
- edge 중간 클릭/탭 = 점 추가
- 핸들 더블클릭(PC) / 롱프레스 500ms(터치) = 점 삭제

## 상태 흐름

```text
drawingMode: null (선택/이동) | "install" | "exclude"

선택/이동 모드 내부:
  idle → (폴리곤 클릭) → selected (툴팁 표시)
  selected → "이동" 선택 → moving (드래그)
  selected → "삭제" 선택 → idle (폴리곤 제거)
  selected → "점 편집" 선택 → editing_vertices (핸들 표시)
  moving → (드래그 종료 or 빈 곳 클릭) → idle
  editing_vertices → (빈 곳 클릭) → idle
  any → (빈 곳 클릭) → idle

모드 전환(그리기 ↔ 선택) 시 진행 중 작업 초기화.
```

## 입력 매핑

| 동작 | PC | 터치 |
|------|-----|------|
| 점 찍기 | 클릭 | 탭 |
| Undo | 우클릭 / 플로팅 버튼 | 플로팅 버튼 |
| 폴리곤 선택 | 클릭 | 탭 |
| 폴리곤 이동 | 마우스 드래그 | 손가락 드래그 |
| 점 이동 | 핸들 드래그 | 핸들 드래그 |
| 점 추가 | edge 중간 클릭 | edge 중간 탭 |
| 점 삭제 | 핸들 더블클릭 | 핸들 롱프레스 (500ms) |
| 선택 해제 | 빈 곳 클릭 | 빈 곳 탭 |

## 시각 표현

- 선택된 폴리곤: 테두리 두께 2배 + 하이라이트 색상
- 이동 중: 반투명 처리
- 점 편집 중: 꼭짓점 핸들 (12px), edge 중간 + 핸들, 드래그 중 핸들 색상 변경
- Undo 버튼: 캔버스 우하단 플로팅, 그리기 중 점 1개 이상일 때만 표시
